# C49 nearest neighbors

| Neighbor | Overlap | Binding boundary |
|---|---|---|
| Cubit | closed-form KRR token mixing | raw-value KRR from the same checkpoint |
| DeltaNet | prediction-error fast-weight update | fixed beta=1 exact delta-rule read |
| Gated DeltaNet | adaptive erasure plus delta update | mandatory if C49 reaches trainable confirmation |
| Titans / test-time memory | surprise or loss-driven memory writes | error-valued memory is not novel by itself |
| Fast Weight Programmers | learned key-value associative memory | C49 must pay rent for the batch least-squares target |

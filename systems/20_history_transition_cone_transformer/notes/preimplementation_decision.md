# C20 pre-implementation decision

Decision: **PASS only to G0 implementation and one hash-locked synthetic GPU
gate.**  Repository records, labels, dev, test and qrels remain forbidden.

The proposal survives exact reduction because a nonorthogonal nonnegative
least-squares reconstruction is not pointwise identical to independent ReLU
retrieval, simplex attention, or unconstrained span projection.  The
one-transition reverse witness makes the sign restriction falsifiable.

The pass is deliberately narrow.  OptNet/differentiable convex layers own the
solver family, LISTA owns unrolling, and relation-aware attention owns edge
representations.  C20 advances only if the positive transition-composition law
beats all nearest controls with matched Transformer capacity, an exchangeable
candidate generator, corruption specificity and exact fallbacks.  Otherwise
it closes before real data.

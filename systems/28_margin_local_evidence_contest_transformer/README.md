# C28 — Margin-Local Evidence-Contest Transformer

C28 keeps C27's token evidence and exactly antisymmetric comparator, but
changes the comparison graph.  Uniform all-pair averaging is replaced by a
fixed continuous kernel over request-standardized D2p score gaps, so evidence
is concentrated on uncertain candidate pairs without naming a top-k boundary.

This is an outcome-isolated train-internal gate, not a novelty claim.  It uses
C27 escrow as a new label-free A0 role and newly generated donors/features.
Uniform C27 aggregation, candidate-only locality, generic evidence locality,
and an additive node head are compute/parameter-matched controls.  Dev/test are
unauthorized.

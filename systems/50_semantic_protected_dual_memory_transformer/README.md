# C50 — Semantic-Protected Dual-Memory Transformer

C50 is a bounded post-C49 formulation.  It keeps the raw semantic KRR read as
one protected memory direction and permits the prequential innovation read to
write only in that direction's orthogonal complement.  It uses the six frozen
C49 checkpoints and only already-open C47-A; no training or fresh role is
authorized at this stage.

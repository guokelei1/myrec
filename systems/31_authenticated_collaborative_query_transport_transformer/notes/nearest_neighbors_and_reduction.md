# C31 nearest-neighbor and reduction boundary

| Neighbor | Shared idea | C31 non-reducible operator tested here |
|---|---|---|
| LLM2Rec | collaborative supervision shapes LM/item representations | strict causal event admission plus one query displacement shared by the entire candidate set |
| GenSAR | semantic and collaborative item identity must coexist | no learned item identifier or dual tower; a low-rank shared LM coordinate deformation is applied to query, history, and candidates |
| ItemRAG | semantic and collaborative item evidence can be combined | no retrieval-time co-purchase channel; only authenticated pre-request history may construct the query transport |
| C02 history hyperadapter | history changes a representation transform | C02 was candidate-conditioned and failed responsiveness; C31 uses one request-level transform and no candidate-conditioned parameters |
| C29 authenticated mediation | provenance is causal and corruption-testable | C29 learned independent candidate scalar deltas; C31 removes that head and enforces one globally coherent query direction |

Required reductions after an A1 pass are frozen as: identity adapter, raw
unauthenticated history with matched trainable capacity, and uniform history
weighting with matched trainable capacity.  Until those survive delayed-B, C31
is a promising primitive rather than an attributable architecture claim.

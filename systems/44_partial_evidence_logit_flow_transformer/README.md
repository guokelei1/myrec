# C44 Partial-Evidence Logit-Flow Transformer

C44 tests one Transformer attention modification after C43: every history
event distributes finite mass across the current candidates plus a null sink,
and the candidate mass is written directly as a zero-sum ranking-logit flow.
History is never pooled into one request profile before the candidate decision.

C44 begins with a data-free synthetic falsifier. It has no repository-data,
train-label, dev, test, or qrels authorization. A passing synthetic gate proves
only operator capacity and rent over matched reductions; it may authorize a
separately locked real-data successor on a new cohort.

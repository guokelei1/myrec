# C24 — Multi-Recurrence Competition Probe

C24 is a train-only architecture-signal gate, not a claimed final method.  It
tests whether a permutation-equivariant candidate-set Transformer can allocate
exact-recurrence evidence among multiple repeated candidates better than the
registered item-only score and an equal-parameter independent calibrator.

Generic listwise self-attention is already known.  A positive outcome would
authorize design of a new recurrence-competition primitive; a negative outcome
would close that information object before further architecture invention.

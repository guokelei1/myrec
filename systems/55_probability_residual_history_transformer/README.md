# C55 — Probability-Residual History Transformer Signal Gate

C55 is a dual-domain signal falsifier for the next architecture, not a paper
system.  It asks whether a history-carrier Transformer can predict the exact
probability residual of a strong non-personalized ranker better than an
equal-capacity candidate/base-only Transformer and matched wrong history.

Both domains use request-standardized base logits.  C55 uses only a frozen
80/20 split of the already authorized C53 fit role; C53-A, reserve, dev, test,
and qrels remain closed.  A positive result would authorize a new architecture
whose internal objective is strong-base complementarity.  A failure would
close loss-target rescue of the current frozen representation family.

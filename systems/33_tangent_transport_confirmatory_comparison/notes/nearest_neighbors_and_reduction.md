# C33 nearest-neighbor boundary

The unprojected C31 transport is trained as the direct matched reduction.  C32
is the same primary on a prior cohort and remains formally closed.  A rejected
mid-BGE insertion is not reused because its paired margin over C32 was
indistinguishable from zero.

BeliefFormer independently uses perpendicular attention residuals inside
Transformer layers (https://openreview.net/forum?id=Ard2QzPAUK), so orthogonal
projection alone is not claimed as novel.  C33 tests whether strict causal
external-memory admission plus one shared personalized ranking query gives the
C32 combination reproducible value beyond its unprojected reduction.

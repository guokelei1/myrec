# C33 Tangent Transport Confirmatory Comparison

C33 is not a new architecture claim.  It is a fresh, paired confirmation of the
C32 spherical-tangent query transport against the capacity- and
initialization-matched unprojected C31 transport.

The fit cohort and every hyperparameter remain fixed.  C33-A, delayed-B, and
escrow are newly selected from label-free reserve requests and exclude C32-A as
well as C32's still-closed delayed-B and escrow.  Both modes train before A
labels open.  A confirmation requires the tangent primary to beat D2p and the
unprojected control under the same seeds, candidate sets, and fold partition.

No dev or test access is authorized.

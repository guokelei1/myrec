# C64 pre-implementation review

Decision: authorize model/tests, label-free data/hash G0, then one fixed
three-seed exposed-fit KuaiSearch probe if G0 passes.

- The experiment is scientifically necessary because almost all later
  candidates kept the LM representation frozen.
- It is not an architecture claim and cannot enter the paper as the proposed
  primitive.
- The exposed C26 fit role is suitable for formulation; a deterministic split
  prevents training/evaluation overlap but is not fresh evidence.
- All 1,200 C26 internal-A labels, delayed roles, Amazon reserve, dev, test, and
  qrels remain closed.
- Candidate sampling is fixed for training efficiency; validation scores full
  candidate sets with common metric code and candidate-hash assertions.
- A failed seed/control/gate terminates this exact probe.  No adaptation-depth
  or optimizer sweep is authorized.

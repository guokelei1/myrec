# C78 preimplementation review

Decision: `authorize_one_data_free_gate`.

C78 is a deliberately simpler successor.  It does not reinterpret C77's
triangle as positive; it promotes the tied query-candidate filter only as an
inherited boundary and tests the independently motivated event-set symmetry.
The hypothesis is falsifiable by the positional, ungated, pairwise, and triadic
set controls on the unchanged surface.

The global novelty ceiling is low because set-equivariant attention is known.
A data-free pass would justify a real architecture-quality probe, not a CCF-A
claim.  A tie against pairwise/triadic set controls closes C78 even if all modes
score well.

# C52 — History-Supported Query-Concept Transformer

C52 tests one internal Transformer intervention after C47--C51 closed fixed
pooled-state confidence/value formulas.  A history KRR projector supplies an
additive bias to the candidate-to-query-token attention logits; the values
remain history-free query/candidate semantic states.  The first gate is an
exposed, dual-domain, zero-training formulation falsifier.  It is not a fresh
result or a paper claim.

The proposal, reduction witness, nearest-neighbour audit, fixed operator,
tests, configuration, and execution lock live in this directory.  Generated
token states, score arrays, and raw reports belong under `artifacts/`; only the
small terminal decision report may be promoted to `reports/`.

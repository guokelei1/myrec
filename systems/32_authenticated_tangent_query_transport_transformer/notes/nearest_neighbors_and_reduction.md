| Neighbor/reduction | Shared part | C32 test-specific difference |
|---|---|---|
| C31 query transport | one authenticated profile moves one shared query | C32 forbids the query-parallel profile component by construction |
| spherical/Riemannian optimization | tangent projection on a unit sphere | projection is not claimed as novel; it is the evidence-write law inside an LM ranker |
| C02 hyperadapter | history changes a representation transform | C32 has no candidate-conditioned parameters or generated matrix; one query tangent is shared by the set |
| generic attention gate | history magnitude can be controlled | C32 has no learned gate or threshold; the zero parallel write follows an exact geometric identity |

adapted-attention tangent operator, and unauthenticated tangent operator.

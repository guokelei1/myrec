# C48 — Signed Influence Consensus Transformer

C48 tests one new internal token-mixing law: a history write is trusted only
to the degree that its signed eventwise KRR influences agree.  It is a
separately scoped successor to the closed C47 candidate-support contraction.

The first gate uses only the already-open C47 train-internal cohorts as a
formulation falsifier.  It cannot produce a confirmatory result or authorize
dev/test.  A fresh C48 architecture gate may be designed only if the fixed
operator beats plain KRR, ordinary attention, and its signed-normalization
nearest control on both exposed domains under the frozen rules.

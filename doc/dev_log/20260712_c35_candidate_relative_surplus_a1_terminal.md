# C35 candidate-relative tangent-surplus A1 terminal

C35 cleanly separated mechanical progress from ranking progress. The
candidate-relative surplus law repaired C34's defining label-free failure:
roughly one quarter of candidate rows abstained exactly and over 90% of active
requests contained both admitted and rejected candidates. All 28 A0 checks
passed across twelve paired GPU fits, so the mechanism was selective,
load-bearing, deterministic, permutation-equivariant, and capacity-matched.

That selectivity did not improve relevance. On the untouched internal-A
cohort, candidate-relative surplus was below D2p for every seed (mean
-0.000567 NDCG@10), tied the absolute-cone predecessor, and was below the
candidate-shared global tangent control by -0.001137. It did beat a generic
candidate-axis softmax by +0.000567, showing that the tangent-surplus law is not
merely ordinary candidate competition; however, this architecture-specific
rent was insufficient and statistically unresolved.

The useful diagnosis is architectural rather than dataset-conditional.
Subtracting each event's candidate-set mean creates abstention, but it also
turns history evidence into a relative contest whose transported direction is
not guaranteed to favor relevant candidates. The global tangent control was
the best registered mode on this cohort and exceeded D2p by +0.000569; because
it was a control rather than the preregistered primary, this is diagnostic only
and cannot be promoted from the exposed cohort.

The next formulation should therefore use a common-plus-contrastive
factorization: retain one authenticated, query-tangent shared history write,
then let candidate-relative evidence control only a centered candidate-specific
deviation. The first gate must test, without outcome labels, that the common
component remains intact, the deviation has both exact abstention and
zero candidate mean, and removal of the common component is a matched nearest
neighbor. No C35 threshold, temperature, centering statistic, or A-cohort
rescue is authorized.

Curated report: `reports/pps_c35_train_gate.json`. Internal-A labels opened only
after A0 passed; delayed-B, escrow, dev, and test remain closed.

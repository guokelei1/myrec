# C63 finite-evidence memory: G0 terminal

C63 replaced C62's duplicating per-slot softmax with an event-wise finite write
budget and explicit NULL remainder.  Slot Attention-style competition,
balanced Sinkhorn transport, C62 standard slots, and pooled memory were trained
as equal-parameter controls on three physical GPUs after hash lock.

All mechanics and gradient contracts passed, but every mode stayed close to
four-way interest chance.  The primary conserved evidence and used all slots,
yet assigned virtually identical NULL mass to useful and nuisance events and
did not react to wrong histories.  This rejects the diagnosis that C62 failed
only because of its normalization axis.

C62--C63 jointly show that final ranking supervision alone does not make a
query-independent latent bottleneck discover addressable preferences, even
when slot competition and evidence conservation are hard-wired.  The memory
family is closed before repository data.  The next step is a separately frozen
full-LM representation-learnability probe: unfreeze pretrained token layers and
ask whether deep query/history/candidate interaction can beat the strong base,
history-free counterpart, and wrong history on exposed train-internal roles.

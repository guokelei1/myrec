# C76 counterfactual layer-trajectory terminal decision

C76 is closed at its data-free gate.  The paired depth trajectory was active,
safe, deterministic, trainable, and capacity matched, but all five modes fit a
train-only candidate nuisance and failed its held-out reversal with supported
accuracy zero.  The trajectory did not isolate evidence identity because
candidate-only coordinates can modulate candidate-history attention and thus
appear inside the history-cut difference.

This narrows the next question.  Full raw-token Q-H-C interaction is still the
only empirically observed positive source, but an admissible write must require
query authentication of the candidate-history token relation.  C77 is the
first post-C76 mechanism update under the terminal C80 budget.

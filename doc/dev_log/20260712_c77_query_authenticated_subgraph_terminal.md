# C77 query-authenticated subgraph terminal decision

C77 is terminal at data-free D1.  It provides the first clean separation of
the C76 nuisance—unsupported candidate-token gradients were structurally zero
and the primary generalized perfectly—but it failed order robustness and tied
the simpler query-candidate token filter.  This is useful boundary evidence,
not an innovation result.

C77 consumes post-C76 architecture update 1/3.  At most two mechanism updates
remain before the C80 terminal retrospective.  A successor may use the simple
query-filter result as a control or architecture boundary, but may not relabel
it as evidence that the C77 triangle was necessary.
